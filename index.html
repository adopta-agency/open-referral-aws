This is the project for develop Human Services Data Specification (HSDS) implementations that run on the Amazon Web Services platform. All work on projects that run on AWS will be coordinated here on this site that runs on Github.

AWS Services

The services that are needed to run an HSDS API on AWS:

  • Amazon EC2 - A Linux Amazon EC2 instance that runs Apache, with PHP 5.3 that runs the API.
  • Amazon RDS - A MySQL Amazon RDS instance that runs the database behind the API.

Tooling

The tooling that is used as part of delivering this project:

  • OpenAPI Specification - An API specification format used to define the request and response model for the API.
  • HSDS PHP Prototype - A current set of PHP scripts that is deployed to support the API.

Communication

The communication channels used as part of this project:

  • Github Issues - A Linux Amazon EC2 instance that runs Apache, with PHP 5.3 that runs the API.
  • Open Referral Slack - A MySQL Amazon RDS instance that runs the database behind the API.

The goal of this project is to develop code and stories around using HSDS on the AWS platform. Right now we are focusing on a simple implementation, but will be looking at serverless approaches using Amazon Lamda services, as well as management using Amazon API Gateway.
