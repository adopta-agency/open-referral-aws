This is the project for develop Human Services Data Specification (HSDS) implementations that run on the Amazon Web Services platform. All work on projects that run on AWS will be coordinated here on this site that runs on Github.

First, what services are needed to run an HSDS API on AWS:

  • Amazon EC2 - A Linux Amazon EC2 instance that runs Apache, with PHP 5.3 that runs the API.
  • Amazon RDS - A MySQL Amazon RDS instance that runs the database behind the API.
